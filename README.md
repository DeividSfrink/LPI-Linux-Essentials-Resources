# 🐧 LPI Linux Essentials (010-160) Study Resources

A curated collection of resources to help you prepare for and pass the **LPI Linux Essentials** certification.

---

## 🗺️ Recommended Study Path
If you're starting from scratch, here’s an efficient way to use these resources:

1. **The Big Picture:** Start with **[Shawn Powers' Intro](https://youtu.be/skTShEHyXfo?si=b4k3XeuYH3G9aaP8)**. It explains the value of the cert and the "pre-calculus" nature of the exam.
2. **The Foundation:** Move to the **Official LPI Learning Materials**. Read the modules and practice the commands in a terminal.
3. **Memorization:** Use the **Quizlet Flashcards** and the **Network Commands Cheat Sheet** to drill command flags and definitions.
4. **Practice & Review:** Go through the **Video Prep Sessions** and **Practice Tests** to see how the concepts appear as exam questions.
